🎮 คู่มือสร้างบอตเตือนงาน Jira + แดชบอร์ด

ฉบับอ่านง่ายที่สุดในโลก — สรุปทีละกริ๊ก ทำตามง่ายเหมือนเล่นเกม!
👋 ยินดีต้อนรับครับ! คู่มือหน้าเว็บนี้เขียนขึ้นมาให้อ่านเข้าใจง่ายเป็นพิเศษ ไม่มีภาษาเขียนโปรแกรมยากๆ มาปน ให้ทำตามทีละขั้นตอนอย่างสบายใจได้เลยครับ!
📌 สารบัญด่านการตั้งค่า (ทำตาม 4 สเต็ปนี้ได้เลย!)
  • 🛠️ ด่านที่ 1: ใช้โปรแกรมช่วยตั้งค่าอัจฉวิยะในเครื่อง (setup-assistant.bat)
  • 🚀 ด่านที่ 2: เอาบอตขึ้นรันออนไลน์บนระบบ Vercel คลาวด์
  • 🔌 ด่านที่ 3: ต่อท่อผูกสายสัญญาณ Webhook ระหว่างระบบ
  • 🏁 ด่านที่ 4: เคาะระฆังเรียกตั๋วงานรอบแรกสุด (First Sync)
🛠️ ด่านที่ 1: ดับเบิ้ลคลิกเพื่อรันสคริปต์อัตโนมัติ

ในโฟลเดอร์หลักของโปรเจกต์คุณ จะมีไฟล์ชื่อ setup-assistant.bat อยู่ ให้ทำตามนี้ครับ:

🚀 วิธีการเริ่มรัน:
  1. ดับเบิ้ลคลิกที่ไฟล์ setup-assistant.bat
  2. หน้าจอสีดำ (CMD) จะเด้งเปิดขึ้นมาช่วยตรวจสอบความพร้อมของคอมพิวเตอร์ของคุณ
  3. ตัวหน้าจอจะเริ่มถามคำถามทีละข้อ และคอยเด้งเปิดเบราว์เซอร์ไปแต่ละหน้าเว็บให้อัตโนมัติ!

💡 วิธีหาค่ามาตอบคำถามทีละหัวข้อ:

ขั้นตอนที่ 1.1
🌐 ลิงก์ฐานข้อมูล Supabase (DATABASE_URL)

เมื่อโปรแกรมถามข้อนี้ และเด้งเปิดหน้าเว็บ Supabase Dashboard ขึ้นมาให้:

  1. คลิกปุ่มสีเขียวคำว่า "Connect" ที่อยู่แถบเมนูด้านบนสุดของโปรเจกต์คุณ
  2. ในหน้าต่างสี่เหลี่ยมที่เด้งขึ้นมา ให้เลือกแท็บ "Direct" ด้านบนสุด
  3. ที่หัวข้อ Connection Method ให้คลิกเลือกตัวเลือก "Transaction pooler"
  4. ที่ช่อง Type ด้านล่าง ให้เลือกตัวเลือกเป็น "URI"
  5. กดคัดลอก (Copy) ลิงก์ยาวๆ ที่ขึ้นต้นด้วย postgres://...
  6. นำมาวางในหน้าจอ CMD และลบคำว่า [YOUR-PASSWORD] ออก แล้วพิมพ์ใส่รหัสผ่านจริงของฐานข้อมูลที่คุณตั้งไว้ลงไปแทน จากนั้นกด Enter
  7. จากนั้น ให้กรอกตั้งชื่อแอดมินและรหัสผ่านที่คุณต้องการใช้ล็อกอินหน้าเว็บ Dashboard ตามใจชอบได้เลยครับ!
ขั้นตอนที่ 1.2
🔑 ข้อมูลการเชื่อมต่อ Jira Cloud (รวม 4 ค่าสำคัญ)

เมื่อโปรแกรมถามข้อนี้ และเด้งเปิดหน้าความปลอดภัย ให้กรอกข้อมูลสำคัญตามนี้ครับ:

  1. โดเมนหลักของ Jira (JIRA_DOMAIN):
    คัดลอกท่อนโดเมนของเบราว์เซอร์คุณ เช่น company-name.atlassian.net (ห้ามใส่ https:// หรือทับ / ต่อท้าย)
  2. อีเมลผู้ดูแลระบบ (JIRA_EMAIL):
    อีเมลที่ใช้ล็อกอินเข้าใช้ระบบ Jira
  3. รหัสผ่านพิเศษ Jira API Token (JIRA_API_TOKEN):
    ในหน้าเว็บความปลอดภัยที่เด้งขึ้นมา ให้กดปุ่ม "Create API token" -> ตั้งชื่อบอต -> กด Copy รหัสยาวๆ ที่ขึ้นต้นด้วย ATATT... มาวาง
  4. รหัสย่อของโปรเจกต์งาน (JIRA_PROJECT_KEY):
    ดูรหัสย่อหน้าเลขตั๋วในระบบ Jira เช่น ถ้าตั๋วงานชื่อ DEV-154 ตัวย่อคือ DEV (พิมพ์ตัวพิมพ์ใหญ่ทั้งหมด)
ขั้นตอนที่ 1.3
💬 ลิงก์แจ้งเตือนกลุ่ม Google Chat Webhook

ข้อนี้ระบบเปิดลิงก์ให้ไม่ได้ ให้คุณเปิดแอป Google Chat ในคอมพิวเตอร์ของคุณเอง แล้วกดเข้าไปในห้องคุยกลุ่มของทีม:

  1. คลิกที่ ชื่อห้องแชทด้านบนสุด -> เลือกเมนู "Apps & integrations" -> เลือก "Webhooks"
  2. คลิกปุ่ม "Add webhook" ตั้งชื่อบอต เช่น AlertBot แล้วกด Save
  3. กดปุ่ม Copy ลิงก์ที่ได้รับมา (จะขึ้นต้นด้วย https://chat.googleapis.com/...) นำมาวางลงในหน้าจอ CMD แล้วกด Enter
ขั้นตอนที่ 1.4
🧠 รหัสคีย์ปัญญาประดิษฐ์ Google Gemini API Key

เมื่อโปรแกรมถามข้อนี้ และเด้งเปิดหน้าเว็บ Google AI Studio ขึ้นมาให้:

  1. คลิกปุ่มรูปกุญแจซ้ายบนคำว่า "Get API key"
  2. คลิกปุ่มสีน้ำเงินคำว่า "Create API key"
  3. เลือกโปรเจกต์ หรือสร้างในโปรเจกต์ใหม่ แล้วกดสร้างคีย์
  4. กดปุ่ม Copy รหัสคีย์ API ที่ขึ้นต้นด้วย AQ. หรือ AL. นำมาวางลงในช่องถามในหน้าจอ CMD แล้วกด Enter
📌 เมื่อตอบคำถามครบแล้ว: ระบบจะสรุปรายชื่อชื่อตัวแปรและข้อมูลคีย์ทั้งหมดทิ้งไว้บนหน้าจอสีดำ ให้คุณลากเมาส์คลุมดำคัดลอกข้อความสรุปทั้งหมดนั้นเก็บใส่ Notepad เอาไว้ เพื่อเตรียมนำไปกรอกบนหน้าเว็บ Vercel คลาวด์ ในสเต็ปถัดไปครับ!
🚀 ด่านที่ 2: เอาโค้ดขึ้นระบบใช้งานจริงออนไลน์ (Deploy to Vercel)

นำโค้ดระบบทั้งหมดขึ้นทำงานออนไลน์ถาวรบนอินเทอร์เน็ตผ่าน GitHub และ Vercel:

  1. เอาโค้ดขึ้น GitHub: สมัครและเปิดเข้าเว็บ GitHub.com กดปุ่มสร้าง New repository ตั้งชื่อโปรเจกต์ที่คุณชอบ เลือกสถานะเป็น Private (ส่วนตัว) เสมอ แล้วจัดการนำโค้ดทั้งหมดในโฟลเดอร์เครื่องคอมของคุณ อัปโหลดขึ้นไปจัดเก็บไว้ให้เรียบร้อย
  2. เชื่อมต่อรันบน Vercel:
    • เปิดเข้าเว็บ Vercel.com ลงชื่อเข้าใช้งานด้วยบัญชี GitHub
    • กดปุ่ม "Add New" -> เลือกหัวข้อ "Project"
    • กดปุ่ม "Import" โปรเจกต์ที่คุณเพิ่งทำในขั้นตอนที่แล้ว
    • เลื่อนหน้าจอลงมาหาหัวข้อตั้งค่าตัวแปรระบบที่ชื่อ "Environment Variables"
    • เปิดโปรแกรม Notepad ที่บันทึกสรุปค่าคีย์ไว้ ก๊อปปี้ชื่อตัวแปร (Key) และค่าข้อมูล (Value) ไปเพิ่มใส่ในตารางทีละช่องจนครบทั้งหมด
    • เมื่อกรอกเสร็จสิ้น คลิกปุ่มสีน้ำเงินคำว่า "Deploy" ด้านล่างสุด
    • รอประมาณ 1-2 นาที เมื่อเสร็จเรียบร้อย ให้กดก๊อปปี้ลิงก์ที่อยู่เว็บของคุณเก็บไว้ (ตัวอย่างลิงก์ เช่น https://jira-googlechat-bot.vercel.app)
🔌 ด่านที่ 3: ต่อท่อส่งข้อมูลการแจ้งเตือน (Webhook Connect)

นำที่อยู่ลิงก์เว็บ Vercel ของคุณไปแจ้งให้ฝั่ง Jira และ Google Chat ได้รับรู้เพื่อส่งสัญญาณข้อมูลหากัน:

3.1 ผูกสัญญาณกับ Jira Cloud (เวลามีการอัปเดตตั๋วงาน)
  1. วิธีเข้าไปหน้าตั้งค่า:
    ล็อกอินเข้าเว็บ Jira ของคุณด้วยบัญชีแอดมิน -> คลิกรูป ฟันเฟือง (Jira Settings) ขวาบน -> เลือกหัวข้อ System -> เลื่อนหาแถบเมนูด้านซ้ายล่างสุด คลิกเลือก Webhooks
    *(หรือเปิดลิงก์นี้โดยตรง: https://[ชื่อโดเมน_Jira_ของคุณ]/plugins/servlet/webhooks)*
  2. คลิกปุ่มมุมขวาบนคำว่า "Create a Webhook"
  3. กรอกข้อมูลตั้งค่าดังนี้:
    • Name: ตั้งชื่อตามชอบ เช่น Sync to Tasky Dashboard
    • Status: เลือกตัวเลือก Enabled
    • URL: วางลิงก์เว็บ Vercel ของคุณ และพิมพ์คำว่า /api/jira/webhook ต่อท้าย (เช่น https://jira-googlechat-bot.vercel.app/api/jira/webhook)
    • JQL Filter: ในช่อง JQL ให้ระบุคำสั่ง project = "คีย์โปรเจกต์ย่อของคุณ" (เช่น project = "DEV")
      ⚠️ สำคัญมาก: เพื่อจำกัดให้ส่งข้อมูลเฉพาะตั๋วงานของโปรเจกต์คุณเท่านั้น ป้องกันบอร์ดอื่นไหลปนครับ
    • Events: เลื่อนลงไปหาหมวดหมู่ที่ชื่อ Issue แล้วติ๊กถูกในช่อง Created, Updated, และ Deleted
  4. เลื่อนลงล่างสุด คลิกปุ่มสีน้ำเงินคำว่า "Create"
👥 การจับคู่รายชื่อสมาชิกทีม (Team Member Mapping) เพื่อส่งแจ้งเตือน

ขั้นตอนที่จะช่วยให้บอตใน Google Chat รู้จักตัวตนของสมาชิกแต่ละคนใน Jira ได้ถูกต้อง:

  1. เปิดเว็บไซต์ Dashboard ของคุณหลังจากติดตั้งขึ้น Vercel เสร็จเรียบร้อยแล้ว
  2. ล็อกอินเข้าสู่ระบบด้วยบัญชีแอดมินที่สร้างไว้ในด่านที่ 1
  3. ไปที่เมนู Team Settings ที่แถบเมนูด้านซ้ายมือ
  4. คลิกเพิ่มชื่อสมาชิกในทีม โดยในช่อง Jira Display Name จะต้องกรอกสะกดให้ตรงกับชื่อผู้รับผิดชอบงานบนระบบ Jira เป๊ะๆ (เช่น Somsak Jaidee) เพื่อให้ระบบแผนผังและบอตเชื่อมต่อข้อมูลและส่งแท็กชื่อตามตัวคนมาทำงานในห้องแชทได้ถูกต้องครับ!
3.2 ผูกสัญญาณกับห้องคุย Google Chat API
  1. เข้าหน้าเว็บ Google Cloud Console
  2. เลือกโปรเจกต์ของคุณ ค้นหาเปิดหน้าบริการที่ชื่อว่า "Google Chat API" แล้วกดคลิกปุ่ม Enable (เปิดใช้งาน)
  3. คลิกปุ่มเมนูการกำหนดค่า "Configuration" ด้านซ้ายมือ กรอกชื่อเล่นของบอตตามใจชอบ
  4. เลื่อนหน้าจอลงมาหาหัวข้อตั้งค่า **Connection settings** ติ๊กเลือกที่หัวข้อ "Webhook URL"
  5. นำลิงก์แอป Vercel ของคุณมาวาง และพิมพ์คำว่า /api/webhook ต่อท้าย (เช่น https://jira-googlechat-bot.vercel.app/api/webhook)
  6. เลื่อนลงด้านล่างสุด คลิกปุ่ม "Save" (บันทึก)
🏁 ด่านที่ 4: เคาะระฆังเรียกตั๋วงานรอบแรกสุด (First Sync)

ขั้นตอนสุดท้ายเพื่อให้ตั๋วงาน Jira ที่มีอยู่เดิมถูกดึงไปจัดระเบียบแสดงบนหน้า แดชบอร์ด ครั้งแรก:

  1. เปิดแท็บเบราว์เซอร์ใหม่ขึ้นมา
  2. พิมพ์เรียกคำสั่งดึงข้อมูลดังนี้: 
    https://[ที่อยู่เว็บ_Vercel_ของคุณ]/api/cron/sync-tickets?secret=[ค่า_CRON_SECRET_ของคุณ]
    *ตัวอย่าง:* https://jira-googlechat-bot.vercel.app/api/cron/sync-tickets?secret=cron_secret_random123
  3. กด Enter หากระบบแสดงผลข้อความคำว่า "sync success" บนหน้าจอ แปลว่าบอตของคุณพร้อมทำงานทุกระบบเรียบร้อยแล้วครับ! 🎉
