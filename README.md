# 🚀 ยินดีต้อนรับสู่ระบบ TaskyBot: Jira & Google Chat Bot + Dashboard

แพลตฟอร์มนี้เป็นระบบรวมบอตแจ้งเตือนตั๋วงาน Jira เข้า Google Chat พร้อมหน้าจอแดชบอร์ดสรุปผลการทำงานของทีม พัฒนาด้วยเทคโนโลยี Next.js (เป็นโครงการแบบ Unified ที่รวมทั้งตัวบอตและแดชบอร์ดไว้ในโปรเจกต์เดียวกันอย่างสมบูรณ์)

---

## 📂 โครงสร้างของไฟล์ในการส่งมอบ (Handoff Package Structure)

เมื่อปลดบล็อกไฟล์ Zip นี้ คุณจะพบโครงสร้างไฟล์ดังนี้:

```text
├── nextjs-app/              # 💻 โค้ดหลักของระบบ (Next.js Application)
│   ├── app/                 # เส้นทางเว็บ API (Webhook, Sync, Login) และหน้า Dashboard
│   ├── components/          # ส่วนประกอบหน้าจอ UI แดชบอร์ด
│   └── setup.js             # สคริปต์ช่วยติดตั้งระบบเบื้องหลัง
├── คู่มือ/                   # 📘 คู่มือแนะนำการติดตั้งและใช้งาน (เปิดด้วยเบราว์เซอร์ได้ทันที)
│   └── 1_คู่มือการติดตั้งระบบ (สำหรับผู้ดูแล).html     # คู่มือสำหรับ Admin ตั้งค่า Cloud/DB/API
├── setup.bat                # ⚡ สคริปต์คลิกเดียวสำหรับเริ่มตั้งค่าตัวแปรระบบ (Windows)
├── vercel.json              # ⏰ ไฟล์กำหนดการทำงานของ Cron Job สรุปงานประจำวัน
└── README.md                # 📝 ไฟล์แนะนำฉบับนี้
```

---

## ⚡ เริ่มต้นติดตั้งใช้งานแบบด่วน (Quick Start)

เพื่อให้บอตเริ่มทำงานได้อย่างรวดเร็ว โปรดทำตามขั้นตอน 3 สเต็ปง่ายๆ ดังนี้ครับ:

### สเต็ปที่ 1: ตรวจสอบความพร้อมของเครื่องคอมพิวเตอร์
* ตรวจสอบว่าในเครื่องคอมพิวเตอร์ของคุณมีโปรแกรม **Node.js (เวอร์ชัน 18 ขึ้นไป)** ติดตั้งอยู่แล้ว
* หากยังไม่มี สามารถดาวน์โหลดได้ฟรีที่ [Node.js Official Website](https://nodejs.org/)

### สเต็ปที่ 2: รันโปรแกรมช่วยติดตั้งอัจฉริยะ (Interactive CLI Setup)
* ดับเบิลคลิกที่ไฟล์ **`setup.bat`** บนเครื่อง Windows ของคุณ
* โปรแกรมหน้าต่างสีดำ (CMD) จะเปิดขึ้นมาและสอบถามค่าตัวแปรสำคัญต่างๆ (เช่น ลิงก์ฐานข้อมูล Supabase, คีย์ Jira API, ลิงก์ Google Chat) โดยระบบจะช่วยตรวจสอบโปรแกรมในเครื่อง สุ่มรหัสความปลอดภัย และสร้างไฟล์ตั้งค่า `.env` ให้อัตโนมัติในโฟลเดอร์ `nextjs-app`

### สเต็ปที่ 3: ศึกษาคู่มือการตั้งค่าฝั่ง Cloud และการใช้งาน
โปรดเปิดไฟล์คู่มือในโฟลเดอร์ `คู่มือ/` ด้วยเว็บเบราว์เซอร์ของคุณเพื่อดูวิธีดำเนินการส่วนที่เหลือ:
1. **วิธีเอาโค้ดขึ้นระบบคลาวด์ Vercel:** อ่านในคู่มือ `คู่มือ/1_คู่มือการติดตั้งระบบ (สำหรับผู้ดูแล).html` (สเต็ปที่ 4)
2. **วิธีผูก Webhooks ฝั่ง Jira และ Google Chat API:** อ่านในคู่มือ `คู่มือ/1_คู่มือการติดตั้งระบบ (สำหรับผู้ดูแล).html` (สเต็ปที่ 5)
3. **วิธีดึงข้อมูลตั๋วงานเก่าครั้งแรก (First Sync):** อ่านในคู่มือ `คู่มือ/1_คู่มือการติดตั้งระบบ (สำหรับผู้ดูแล).html` (สเต็ปที่ 6)

## 🔒 นโยบายความปลอดภัยและข้อควรระวัง
* **ห้ามแชร์ไฟล์ `.env` หรือ `service-account.json` ให้กับบุคคลอื่นที่ไม่เกี่ยวข้อง** เนื่องจากไฟล์เหล่านี้เปรียบเสมือนกุญแจสำคัญในการเข้าถึงข้อมูลตั๋วงาน Jira ของบริษัทและห้องแชทขององค์กร
* สิทธิ์การใช้งานบอต Google Chat จะถูกจำกัดไว้ที่โดเมนอีเมลขององค์กรคุณเป็นหลักตามสิทธิ์ของ Google Chat API
