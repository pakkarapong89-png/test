## 🛠️ ขั้นตอนที่ 1: ใช้โปรแกรมช่วยตั้งค่าอัจฉริยะ (ดับเบิ้ลคลิกเดียวจบ)
ในโฟลเดอร์โปรเจกต์ของคุณ จะมีไฟล์ชื่อ **`setup-assistant.bat`** อยู่ ให้ทำตามนี้เลยครับ:

1. **ดับเบิ้ลคลิกเปิดไฟล์ `setup-assistant.bat`**
2. หน้าจอสีดำ (CMD) จะเปิดขึ้นมาทักทายคุณ และตรวจสอบความพร้อมของคอมพิวเตอร์คุณโดยอัตโนมัติ
3. จากนั้น โปรแกรมจะคอยถามคำถามทีละข้อ และคอยเด้งเปิดหน้าเว็บขึ้นมาให้คุณโดยที่คุณไม่ต้องไปเปิดเอง! ให้ทำตามวิธีหาคีย์ดังนี้ครับ:

### 📡 1.1 ลิงก์ฐานข้อมูล Supabase (DATABASE_URL)
*เมื่อสคริปต์เด้งหน้าเว็บ Supabase ขึ้นมาให้ ให้กดเข้าไปที่หน้าโปรเจกต์ของคุณแล้วทำดังนี้:*
* **วิธีเอาค่า:**
  1. มองไปที่แถบเมนูด้านบนสุดของโปรเจกต์ คลิกที่ปุ่มสีเขียวคำว่า **"Connect"**
  2. ในหน้าต่างสี่เหลี่ยมที่เด้งขึ้นมา ให้กดเลือกแท็บ **"Direct"** ด้านบนสุด
  3. หัวข้อ Connection Method ติ๊กเลือก **"Transaction pooler"**
  4. ช่อง Type ให้เลือกเป็น **"URI"**
  5. กดปุ่ม **Copy** ลิงก์ที่ขึ้นต้นด้วย `postgres://...` ยาวๆ มาวางในช่องคำถามที่ CMD
  6. **⚠️ สำคัญมาก:** ลิงก์ที่ก๊อปมาจะมีคำว่า `[YOUR-PASSWORD]` อยู่ในนั้น ให้ลบคำนี้ออกแล้วพิมพ์**รหัสผ่านฐานข้อมูลจริง**ที่คุณตั้งไว้ตอนสมัคร Supabase ใส่เข้าไปแทนนะครับ จากนั้นกด **Enter**
  7. สคริปต์จะสร้างบัญชีแอดมินสำหรับหน้าเว็บบอร์ดแดชบอร์ด ให้ตั้งชื่อ (Username) และ รหัสผ่าน (Password) ตามใจชอบได้เลยครับ!

### 🔑 1.2 ข้อมูลการเชื่อมต่อ Jira Cloud (รวม 4 ค่าสำคัญ)
*เมื่อระบบถามถึงตัวแปรของ Jira ให้เตรียมค่าเหล่านี้มาตอบคำถามครับ:*

* **1.2.1 โดเมนหลักของ Jira (`JIRA_DOMAIN`)**
  * **วิธีดู:** ให้เปิดหน้าเว็บ Jira ที่คุณใช้งานปกติ แล้วมองไปที่แถบที่อยู่เว็บด้านบน (URL)
  * **สิ่งที่ต้องเอามา:** คัดลอกเฉพาะท่อนที่เป็นชื่อโดเมน เช่น `company-name.atlassian.net` (ห้ามพิมพ์ `https://` หรือเครื่องหมายทับ `/` ต่อท้ายเด็ดขาด ไม่งั้นบอตจะต่อสายไม่ติดนะครับ!)
  
* **1.2.2 อีเมลผู้ดูแลระบบ (`JIRA_EMAIL`)**
  * **สิ่งที่ต้องเอามา:** อีเมลที่คุณใช้ในการล็อกอินเข้าทำงานในระบบ Jira ขององค์กรหรือโปรเจกต์นั้นๆ

* **1.2.3 รหัสผ่านพิเศษสำหรับบอต (`JIRA_API_TOKEN`)**
  * *เมื่อสคริปต์เด้งเปิดหน้าความปลอดภัยของ Atlassian ขึ้นมาให้:*
  * **วิธีสร้าง:**
    1. คลิกปุ่มสีน้ำเงินคำว่า **"Create API token"**
    2. พิมพ์ตั้งชื่อป้ายกำกับอะไรก็ได้ตามชอบ เช่น `Jira-GoogleChat-Bot` แล้วกดปุ่ม Create
    3. กดปุ่ม **Copy** รหัสผ่านพิเศษที่ได้มา (จะขึ้นต้นด้วยตัวอักษร **"ATATT..."** ยาวๆ) นำมาวางลงในช่องถามในหน้าจอ CMD ทันทีแล้วกด Enter
    4. **⚠️ ข้อควรระวัง:** รหัสนี้จะแสดงให้เห็นเพียงแค่ครั้งเดียวตอนกดสร้าง หากปิดหน้าต่างไปแล้วจะไม่สามารถดูรหัสเดิมได้อีก ควรบันทึกรหัสสำรองเก็บไว้ใน Notepad ด้วยนะครับ

* **1.2.4 รหัสย่อของโปรเจกต์งาน (`JIRA_PROJECT_KEY`)**
  * **วิธีดู:** เข้าไปดูรหัสตั๋วงานใน Jira ของทีมคุณ เช่น ถ้าตั๋วงานมีชื่อรหัสว่า `DEV-12`, `DEV-230`, หรือ `TES-99`
  * **สิ่งที่ต้องเอามา:** ให้คัดลอกเอาเฉพาะตัวอักษรย่อด้านหน้า เช่น **`DEV`** หรือ **`TES`** (พิมพ์ตัวใหญ่ทั้งหมด) นำมากรอกใส่ช่องนี้ได้เลยครับ คีย์นี้จะช่วยให้บอตไปดึงงานถูกกล่องนั่นเองครับ!

### 🧠 1.3 รหัส Gemini API Key (สมองบอต AI)
*เมื่อสคริปต์ถามคำถามนี้ และเด้งเปิดหน้าเว็บ Google AI Studio ขึ้นมาให้:*
* **วิธีเอาค่า:**
  1. คลิกปุ่มคำว่า **"Get API key"** ที่แถบเมนูด้านซ้ายบน (ไอคอนรูปกุญแจ)
  2. คลิกปุ่มสีน้ำเงินคำว่า **"Create API key"**
  3. เลือกโปรเจกต์ หรือกดเลือกสร้างในโปรเจกต์ใหม่ แล้วกดสร้างรหัสคีย์
  4. รหัสจะขึ้นต้นด้วยตัวย่อ **"AQ."** หรือ **"AL."** ให้กดปุ่ม **Copy** นำมาวางลงในช่องคำถามในหน้าจอ CMD ทันทีแล้วกด **Enter**

### 💬 1.4 ลิงก์แจ้งเตือน Google Chat Webhook URL
*ช่องนี้ระบบเปิดหน้าเว็บให้ไม่ได้เพราะเป็นห้องแชทส่วนตัว ให้คุณเปิดแอป Google Chat ของคุณขึ้นมาเองครับ:*
* **วิธีเอาค่า:**
  1. เข้าห้องแชท (Space) กลุ่มของทีมคุณที่ต้องการให้บอตตัวนี้เข้าไปส่งการแจ้งเตือนงาน
  2. คลิกที่ **ชื่อห้องแชทด้านบนสุด** -> เลือกเมนู **"Apps & integrations" (แอปและการบูรณาการ)** -> เลือก **"Webhooks"**
  3. คลิกปุ่ม **"Add webhook" (เพิ่มเว็บฮุค)**
  4. ตั้งชื่อชื่อโปรไฟล์ของบอตตามใจชอบ (เช่น `Tasky Alert Bot`) แล้วกดปุ่ม **Save (บันทึก)**
  5. กดปุ่ม **Copy** ลิงก์ยาวๆ ที่ได้รับมา (ขึ้นต้นด้วย `https://chat.googleapis.com/...`) นำมาวางลงในหน้าจอ CMD แล้วกด **Enter**

> [!TIP]
> เมื่อกรอกคำถามอื่นๆ จนครบแล้ว ตัวโปรแกรมจะสร้างไฟล์สำหรับรันระบบให้เองอัตโนมัติ และจะทำการ**สรุปคีย์ข้อความทั้งหมดพิมพ์ทิ้งไว้บนหน้าจอ** ให้คุณคัดลอกข้อความสรุปนั้นเก็บลงใน Notepad ไว้ เพื่อเตรียมนำไปใช้ในขั้นตอนถัดไปครับ!

## 🚀 ขั้นตอนที่ 2: เอาโค้ดขึ้นอินเทอร์เน็ตใช้งานจริง (Deploy to Vercel)
ขั้นตอนนี้เราจะเอาโค้ดทั้งหมดขึ้นทำงานออนไลน์ เพื่อให้บอตคอยฟังข้อมูลได้ตลอด 24 ชั่วโมงแม้คุณจะปิดเครื่องคอมพิวเตอร์ครับ:

1. **เอาโค้ดขึ้น GitHub:** สร้าง Repository ใหม่บนเว็บ GitHub.com ตั้งชื่อตามต้องการ เลือกสถานะเป็น **Private (ส่วนตัว)** เสมอ แล้วอัปโหลดโค้ดทั้งหมดขึ้นไปเก็บไว้
2. **เอาโค้ดขึ้น Vercel:**
   * ล็อกอินเข้าเว็บ **Vercel.com** ด้วยบัญชี GitHub ของคุณ
   * กดปุ่ม **"Add New"** -> เลือก **"Project"**
   * กดปุ่ม **"Import"** ตัวโปรเจกต์ที่คุณเพิ่งอัปโหลดขึ้น GitHub เมื่อสักครู่
   * เลื่อนหน้าจอลงมาหาแถบตั้งค่าที่ชื่อ **Environment Variables**
   * เปิด Notepad ที่เราบันทึกค่าสรุปจากข้อ 1 ขึ้นมา ก๊อปปี้ชื่อตัวแปร (Key) และค่าข้อมูล (Value) ไปกรอกลงในช่องทีละคู่จนครบ
   * เมื่อกรอกครบแล้ว คลิกปุ่มสีน้ำเงินคำว่า **"Deploy"**
   * รอระบบโหลด 1-2 นาที เมื่อเสร็จเรียบร้อย Vercel จะแสดงผลยินดีด้วย พร้อมลิงก์ที่อยู่เว็บสาธารณะของคุณมาให้ (เช่น <code>https://jira-googlechat-bot.vercel.app</code>) ให้ก๊อปปี้ลิงก์นี้เก็บไว้ใช้งานครับ

## 🔌 ขั้นตอนที่ 3: ต่อท่อส่งข้อมูลการแจ้งเตือน (Webhook Connect)
ขั้นตอนนี้จะทำให้บอตบนคลาวด์ของเราเชื่อมต่อสายสื่อสารกับ Jira และห้องแชทได้ครับ:

### 3.1 ผูกสัญญาณกับ Jira (เวลามีการอัปเดตตั๋วงาน)
เพื่อให้ตั๋วงานบน Jira ซิงก์ข้อมูลกลับมาหาตัวแดชบอร์ดของเราทันทีเมื่อมีคนสร้างงานใหม่ แก้ไขงาน หรือปิดงาน:
1. **วิธีเข้าไปหน้าเว็บฮุค:**
   * ล็อกอินเข้าเว็บ Jira ในฐานะ **Admin** ของระบบ
   * คลิกที่รูป **ฟันเฟือง (Jira Settings)** มุมขวาบนสุด -> เลือกหัวข้อ **System**
   * มองหาแถบเมนูด้านซ้าย เลื่อนลงล่างสุดในหมวด Advanced แล้วคลิกที่เมนู **Webhooks**
   * *(หรือหากขี้เกียจหาเมนู ให้พิมพ์เปิดลิงก์นี้ตรงๆ ได้เลยครับ: `https://[โดเมน_Jira_ของคุณ]/plugins/servlet/webhooks`)*
2. คลิกปุ่มสีน้ำเงินคำว่า **"Create a Webhook"** ที่มุมขวาบน
3. กรอกรายละเอียดดังนี้:
   * **Name:** ตั้งชื่อที่ต้องการจำง่าย เช่น `Sync to Tasky Dashboard`
   * **Status:** เลือกเป็น `Enabled` (เปิดใช้งาน)
   * **URL:** นำที่อยู่ลิงก์เว็บ Vercel ของคุณมากรอก และพิมพ์คำว่า `/api/jira/webhook` ต่อท้ายเสมอ
     * *ตัวอย่าง:* `https://jira-googlechat-bot.vercel.app/api/jira/webhook`
   * **JQL Filter:** ในช่องใส่สูตร JQL ด้านล่างช่อง URL ให้พิมพ์สูตรกรองดังนี้:
     * `project = "รหัสย่อโปรเจกต์ย่อของคุณ"` (เช่น `project = "DEV"`)
     * *⚠️ สำคัญมาก: หากไม่ใส่ข้อนี้ เวลาทีมอื่นในบอร์ดอื่นๆ อัปเดตงาน ข้อมูลขยะจะไหลทะลักเข้ามาปนเปในระบบแดชบอร์ดของเราครับ!*
   * **Events (เหตุการณ์ที่ต้องการแจ้งเตือน):** ให้เลื่อนหน้าจอลงมาหาหัวข้อใหญ่ชื่อ **Issue** แล้วคลิกติ๊กถูกที่ช่อง 3 ช่องนี้:
     * **Created** (เมื่อมีการสร้างตั๋วงานใหม่)
     * **Updated** (เมื่อมีการย้ายสถานะ, เปลี่ยนชื่อคนทำ, หรือแก้ไขข้อความตั๋วงาน)
     * **Deleted** (เมื่อมีการลบตั๋วงาน)
4. เลื่อนลงไปด้านล่างสุด แล้วคลิกปุ่ม **"Create"**

### 📋 การจับคู่รายชื่อสมาชิกทีม (Team Member Mapping) เพื่อส่งแจ้งเตือนรายบุคคล
เพื่อให้บอต AI ฉลาดพอที่จะรู้ว่าตั๋วงานใน Jira ที่ระบุชื่อคนทำคนนั้นๆ ตรงกับไอดี Google Chat คนไหนเวลาแจ้งเตือนรายบุคคล:
1. เมื่อทำการติดตั้งโปรเจกต์และรันแดชบอร์ดขึ้น Vercel เรียบร้อยแล้ว ให้เปิดหน้าเว็บไซต์ Dashboard ของคุณขึ้นมา
2. ล็อกอินด้วยบัญชีแอดมินที่คุณสร้างไว้ในขั้นตอนที่ 1
3. เข้าเมนู **Team Settings (หรือตั้งค่าพนักงาน)** ด้านซ้ายมือ
4. คลิกเพิ่มสมาชิกทีมใหม่ โดยกรอกชื่อ **Jira Display Name** ให้สะกดตรงกันกับชื่อผู้รับผิดชอบบน Jira เป๊ะๆ (เช่น `Somsak Jaidee`) และทำการใส่ลิงก์ webhook หรืออีเมลกลุ่มเพื่อให้บอตสามารถส่งข้อความไปแท็กตามคนมารับงานได้ถูกต้องครับ!

### 3.2 ผูกสัญญาณกับห้องแชท Google Chat App (เวลาคนพิมพ์ @Mention สั่งงานบอต)
1. เข้าหน้าเว็บ **[Google Cloud Console](https://console.cloud.google.com/)** เลือกโปรเจกต์ของคุณ
2. ค้นหาเปิดบริการที่ชื่อ **"Google Chat API"** แล้วคลิกเปิดใช้งาน (**Enable**)
3. คลิกเมนูตั้งค่า **"Configuration"** ด้านซ้ายมือ กรอกชื่อบอตและรูปโปรไฟล์ให้สวยงาม
4. เลื่อนลงมาหาหัวข้อ **Connection settings** ติ๊กเลือกที่ **"Webhook URL"**
5. นำลิงก์แอป Vercel ของคุณมากรอก และพิมพ์ต่อท้ายด้วยคำว่า `/api/webhook` เสมอ
   * *ตัวอย่าง:* `https://jira-googlechat-bot.vercel.app/api/webhook`
6. กดปุ่ม **"Save" (บันทึก)** ด้านล่างสุด

## 🏁 ขั้นตอนที่ 4: ดึงข้อมูลงานครั้งแรกเข้าแดชบอร์ด (First Sync)
เพื่อให้หน้าบอร์ดแสดงผลรายชื่อและดึงงาน Jira ที่มีอยู่เดิมมาเก็บไว้ในระบบเป็นครั้งแรก:
1. เปิดหน้าแท็บเบราว์เซอร์ใหม่ขึ้นมา
2. พิมพ์เปิดลิงก์สั่งงานดังนี้ลงบนช่องที่อยู่เว็บ:
   `https://[ที่อยู่เว็บ_Vercel_ของคุณ]/api/cron/sync-tickets?secret=[ค่า_CRON_SECRET_ของคุณ]`
   * *ตัวอย่างลิงก์:* `https://jira-googlechat-bot.vercel.app/api/cron/sync-tickets?secret=cron_secret_random123`
3. หากแสดงผลข้อความระบุว่า `sync success` แปลว่าระบบทำงานเสร็จสิ้นแล้ว!

**🎉 ตอนนี้บอตและหน้า Dashboard ของคุณก็พร้อมรันใช้งานจริง แดนซ์เตือนข้อมูลงานค้างได้ยาวๆ แบบสุดเท่แล้วครับ!**
