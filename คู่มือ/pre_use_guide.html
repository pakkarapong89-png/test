🎒 คู่มือเตรียมตัวก่อนเริ่มติดตั้งบอต

เช็คลิสต์เตรียมของให้ครบถ้วนก่อนลงมือ — ทำได้ไวใน 5 นาที ไม่สะดุดกลางคัน!
⚠️ กรุณาเตรียมการตามคู่มือนี้ก่อนเริ่มใช้งาน: การลงทะเบียนสิทธิ์และการสมัครบัญชีล่วงหน้า จะช่วยปูทางให้การทำตามขั้นตอนในสคริปต์ setup-assistant.bat ทำงานได้ราบรื่นทันที โดยไม่ต้องปิดสคริปต์ไปขออนุมัติจากแอดมินหรือรอสมัครบัญชีให้เสียเวลาครับ
📦 ด่านที่ 1: เช็คลิสต์สมัครบัญชีผู้ใช้งาน (สมัครฟรีทั้งหมด)

ให้เปิดเบราว์เซอร์ของคุณขึ้นมาและทำการสมัครใช้งานรวมถึงล็อกอิน (Login) บัญชีเหล่านี้เตรียมไว้ล่วงหน้าครับ:

✓
1. บัญชี Jira Cloud ของทีม:
บอร์ดงานหลักของคุณ เช่น mycompany.atlassian.net เพื่อใช้เป็นฐานตั้งต้นดึงงาน
✓
2. บัญชี Google (Gmail):
สำหรับใช้สร้างคีย์ Gemini API บนหน้าจอ Google AI Studio
✓
3. บัญชี Supabase.com:
ฐานข้อมูลระบบคลาวด์สำหรับเก็บบันทึกประวัติตั๋วงานและรายชื่อสมาชิกในโปรเจกต์
✓
4. บัญชี GitHub.com:
สำหรับใช้จัดเก็บและฝากซอร์สโค้ดของระบบบอตให้ทำงานในที่ปลอดภัย
✓
5. บัญชี Vercel.com:
แพลตฟอร์มสำหรับเปิดหน้าเว็บไซต์ Dashboard และเซิร์ฟเวอร์ตอบรับบอตตลอด 24 ชั่วโมง
🔑 ด่านที่ 2: ตรวจสอบระดับสิทธิ์การเข้าถึง (Permissions)

เพื่อป้องกันไม่ให้เจอปัญหาคีย์ไม่ทำงานหรือเซฟไม่ได้ในภายหลัง ให้เช็คสิทธิ์ในโปรเจกต์ของตัวคุณเองตามนี้ก่อนครับ:

สิทธิ์ฝั่งระบบ Jira Cloud:
  • คุณต้องสามารถเข้าถึงการตั้งค่าตั๋วงานในโปรเจกต์นั้นๆ ได้ และมีสิทธิ์เป็น **Admin** หรือมีสิทธิ์ในการสร้าง Webhook
  • 💡 วิธีเช็คด่วน: ลองเปิดเข้าลิงก์ https://[โดเมน_Jira_ของคุณ]/plugins/servlet/webhooks หากสามารถเข้าได้แสดงว่าผ่าน!
  • (หากระบบล็อกสิทธิ์เข้าไม่ได้: แนะนำให้แจ้งผู้ดูแลระบบไอทีของบริษัทเพื่อเปิดสิทธิ์หรือก๊อปปี้ URL บอตท้ายการติดตั้งส่งไปให้ทางแอดมินแอดเชื่อมระบบให้แทนครับ)
สิทธิ์ฝั่ง Google Chat:
  • ในห้องคุยกลุ่มของทีม (Space) คุณจะต้องมีสิทธิ์คลิกที่ปุ่มตัวเลือกหัวข้อ "Apps & integrations" เพื่อสร้างทางเชื่อมต่อ **Webhook** ได้
  • (หากสร้างไม่ได้: ร้องขอให้หัวหน้าทีมหรือผู้ดูแลห้องแชทกดสร้างตามด่านที่ 1.3 และส่งคีย์ลิงก์มาให้คุณนำไปตอบคำถามแทน)
📝 ด่านที่ 3: กระดาษโน้ตจดข้อมูลสำคัญรอป้อนระบบ

เปิดโปรแกรม Notepad บนเครื่องของคุณขึ้นมารอ แล้วไปหาค่าเหล่านี้มาก๊อปปี้วางจดรอไว้ได้เลยครับ:

1. ลิงก์ฐานข้อมูล Supabase (DATABASE_URL)

สร้างโปรเจกต์ใหม่บนเว็บ Supabase -> คลิกปุ่มเขียว "Connect" ด้านบนสุด -> แท็บ "Direct" -> ติ๊กเลือก "Transaction pooler" -> เลือกประเภทเป็น "URI" -> กด Copy ลิงก์ที่ขึ้นต้นด้วย postgres://... มาแปะใน Notepad และพิมพ์แทนที่คำว่า [YOUR-PASSWORD] ด้วยรหัสผ่านจริงของคุณ

2. ชื่อโดเมนและคีย์ย่อโครงการ Jira (JIRA_DOMAIN & JIRA_PROJECT_KEY)

JIRA_DOMAIN: คัดลอกลิงก์หน้าบอร์ด Jira ของคุณมาเฉพาะชื่อโดเมน เช่น company.atlassian.net
JIRA_PROJECT_KEY: รหัสตัวย่อบนตัวเลขตั๋วงาน เช่น งานชื่อย่อ TES-125 คีย์ย่อคือ TES

3. ลิงก์ห้องแชท Webhook URL (GOOGLE_CHAT_WEBHOOK_URL)

ลิงก์ยาวๆ ที่ก๊อปปี้มาจากการสร้าง Webhook ในหัวข้อ Google Chat (เช่น https://chat.googleapis.com/v1/spaces/...)

4. สมอง AI ปัญญาประดิษฐ์ (GEMINI_API_KEY)

กดสร้างจากหน้าเว็บ Google AI Studio -> คัดลอกรหัสคีย์เก็บมาวางรอไว้ (ตัวรหัสจะขึ้นต้นด้วยตัวย่อ **`AQ.`** หรือ **`AL.`**)
